ADD YOUR PHOTOS HERE
====================

Instructions:
1. Add your photo files to this folder
2. Name them something simple like:
   - ashley-memory-1.jpg
   - ashley-memory-2.jpg
   - ashley-memory-3.jpg
   etc.

3. Reference them in the code as:
   const photoUrl = "/photos/ashley-memory-1.jpg";

Supported formats:
- JPG/JPEG
- PNG
- WebP
- GIF

Recommended:
- Compress images to 500KB-1MB for faster loading
- Use square or near-square dimensions for best Polaroid display
- 1024px x 1024px is a good target size
